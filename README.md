# Credit-Spread-Regime-Model-CSRM
Credit Spread Regime Model (CSRM) is a deterministic macro system that uses credit proxies, Treasury spreads, VIX, and optional FRED data to classify market regimes (Risk-On/Neutral/Risk-Off). It tracks stress, trend, divergence, and persistence to generate explainable signals for allocation, risk management, and macro decision-making.
